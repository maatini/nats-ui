# NATS Nexus UI

A professional management UI for NATS.

## Getting Started

### 1. Start NATS Server
You can start a local NATS server with JetStream enabled using Docker Compose:

```bash
docker-compose up -d
```

### 2. Run the UI
Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the dashboard.
